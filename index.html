Choose a volunteer's ID number from the "Test Subject ID No.:" drop-down menu. Once the ID number is selected the following charts will be updated with that test subject's results. The subject's demographic info, the top 10 bacteria cultures found, the number of bacteria cultures per sample, and their washing frequency will all update once and ID is chosen. When you hover your mouse over each type (OTU ID) it will show all the bacteria that make up each culture on the first two charts.
